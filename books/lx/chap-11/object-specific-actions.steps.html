Object-Specific Actions Demo Steps

Goal: Create Object-Specific Action with name: New SPL Contact on Account object to :

  • Create a new Contact when you are in Account object using this Action

Steps:

  1. Create New Object-Specific Action New SPL Contact for the Account Object with target object Contact
  2. Create Actions Layout for this Action with these fields for new Contact record:
    • Name
    • Email
    • Phone
  3. Add this Action to Page Layouts of the Account

Notes:

Adding too many fields to an action layout can impact user efficiency. Recommended maximum is 8 fields.

To reduce the number of fields, you can set predefined values for fields that need a value, but won't often be edited.

You can safely remove those fields from the layout. Set predefined values from the action detail page.
